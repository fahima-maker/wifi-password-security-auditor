# Wi-Fi Password Security Auditor

## Description
The Wi-Fi Password Security Auditor is a Python application that evaluates the strength of a Wi-Fi password using common security rules.

## Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers and special characters
- Identifies common weak passwords
- Provides security recommendations
- Generates a secure random password suggestion

## Technologies Used
- Python
- Regular Expressions (re)
- Random Module

## How to Run
python wifi_password_security_auditor.py
